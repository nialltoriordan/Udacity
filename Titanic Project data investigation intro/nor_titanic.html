I'm not too familar with the Titanic disaster, so I'm not really sure what questions I should be asking. I've heard that it was rumoured that steerage class passengers were locked in their cabins, so it might be interesting to see what the survivorship ratio was by ticket class. I've also heard that because of the 'women and children first' rule, some men were mocked for getting into the lifeboats. It might be interesting to divide the survivors into three groups (men, women and children) and see what the survivorship ratios are for those groups. Lastly, I plan to divide the passengers by age. This is mostly an excercise to divide the survivors into bins and see what the ratios were. So in summary, the questions I plan to ask are :

What are the survivorship ratios by passenger class?

What are the survivorship ratios for men, women, and children?

What are the survivorship ratios by age?

Wrangling and exploring.

The dataset is relatively small so it's possible to view it as a spreadsheet and draw some conclussions. Every passenger has a numeric passenger class attribute. In answering the first question, I can use the entire dataset, but I suspect it may be helpfull to create a dataset of survivors too.

The next two questions could be more difficult to answer because not every passenger has an age attribute. It might be possible to still classify some of these as men, women or children, using their title (e.g. Master) for example, but it's probably best to exclude these altogether. 176 passengers have no age attribute which works out at just under 20% of the 891 passengers, so we should still have a representative sample. For these two questions it'll be important to remember to use the sample standard deviation. I'll create new subsets of data for passengers and survivors with an age attibute. For larger datasets it would probably be impractical to do this, because I suspect it wouldn't be much more diffult to answer the questions without creating a new subset, but in this instance I think it'll be easier and more readable to create the subsets.

def count_classes(my_dataframe):
    
    first_class_count = my_dataframe['Pclass'][(my_dataframe['Pclass']==1)].sum()
    second_class_count = (my_dataframe['Pclass'][(my_dataframe['Pclass']==2)].sum())/2
    third_class_count = (my_dataframe['Pclass'][(my_dataframe['Pclass']==3)].sum())/3
    
    return first_class_count, second_class_count, third_class_count

#Get type of passengers, then survivors.

def count_types(my_dataframe, adult_age = 18.0):
    
    male_count = len(my_dataframe['Sex'][(my_dataframe['Sex']=='male') & (my_dataframe['Age']>=adult_age) ])
    female_count = len(my_dataframe['Sex'][(my_dataframe['Sex']=='female') & (my_dataframe['Age']>=adult_age) ])
    child_count = len(my_dataframe['Sex'][(my_dataframe['Age'] < adult_age) ])
                                          
    return male_count, female_count, child_count
